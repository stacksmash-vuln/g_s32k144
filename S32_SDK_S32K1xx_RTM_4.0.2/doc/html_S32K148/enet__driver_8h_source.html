570  ENET_RX_ALMOST_EMPTY, /*<! Value, in 64-bit words, of the receive FIFO almost empty threshold. When the FIFO
571  level reaches this level and the end-of-frame has not been received for the frame
572  yet, the core receive read control stops FIFO read. A minimum value of 4 should be
573  set. */
574  ENET_RX_ALMOST_FULL, /*<! Value, in 64-bit words, of the receive FIFO almost full threshold. When the FIFO level
575  comes close to the maximum, so that there is no more space for at least RX_ALMOST_FULL
576  number of words, the MAC stops writing data in the FIFO and truncates the received frame
577  to avoid FIFO overflow. A minimum value of 4 should be set. */
578  ENET_TX_SECTION_EMPTY,
