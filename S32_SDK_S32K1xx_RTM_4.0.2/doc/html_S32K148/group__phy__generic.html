Generic PHY
Ethernet Transciever (PHY) » PHY Framework

Detailed Description

Generic PHY driver for 10/100/1000 PHYs following IEEE 802.3-Clause 22.

Supported devices

This driver supports generic devices, supporting Clause 22 IEEE standardized registers. This includes typical 10/100/1000 PHYs like 100BASE-TX or 1000BASE-T

Unsupported API

The following functions are not supported by this driver:

  • PHY_SleepAbortReject
  • PHY_Sleep
  • PHY_Wakeup
  • PHY_SelectTestMode
  • PHY_EnableInterrupts
  • PHY_HandleInterrupt
