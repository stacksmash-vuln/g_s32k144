47  /*
48  * The IV for this implementation is 96 bits to give the most flexibility.
49  *
50  * Some protocols may have unique per-invocation inputs that are not
51  * 96-bit in length. For example, IPsec may specify a 64-bit nonce. In
52  * such a case, it is up to the protocol document to define how to
53  * transform the protocol nonce into a 96-bit nonce, for example by
54  * concatenating a constant value.
55  */
